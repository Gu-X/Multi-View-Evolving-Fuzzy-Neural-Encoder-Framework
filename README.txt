# Multi-View-Evolving-Fuzzy-Neural-Encoder-Framework

The Python code for the research paper titled "Self-Organising Explainable Multi-View Representation Learning for Remote Sensing Scene Classification".

This work is described in:
===============================================================================================================================
Xiaowei Gu, Abdulrahman Kerim, Jinghao Zhang, Jungong Han, Qiang Shen, Peter M Atkinson, Ce Zhang,
Self-Organising Explainable Multi-View Representation Learning for Remote Sensing Scene Classification,
Applied Soft Computing, 2026, 114579,ISSN 1568-4946, https://doi.org/10.1016/j.asoc.2026.114579.
===============================================================================================================================
Please cite the paper above if this code helps. 
Programmed by Xiaowei Gu. For any queries about the code, please contact Dr. Xiaowei Gu: xiaowei.gu@surrey.ac.uk

The datasets used in the main file are not provided in this repo. The links to the datasets used for experiments can be found in this paper.
